## тЭУ Troubleshooting

- **Solution**: Check image size (max 5MB), use JPG/PNG format

- **Solution**: Allow location permissions in browser settings

- **Solution**: Check internet connection, try again in a few seconds

- **Solution**: Allow pop-ups for KhetGo in browser settings

- **Solution**: Check notification permissions in browser/device settings
