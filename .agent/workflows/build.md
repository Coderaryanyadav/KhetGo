# Build Workflow

This workflow builds the KhetGo project for production use.

1. Ensure all dependencies are installed:
// turbo
```bash
npm install
```

2. Run the production build:
// turbo
```bash
npm run build
```

3. Preview the build:
```bash
npm run preview
```

4. Verify build artifacts in the `dist` directory.
